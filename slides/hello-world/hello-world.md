# Hello, World!

Marp では **Markdown** でスライドを書けます。

- 見出しは `#` や `##`
- リストは `-` や `*`
- 区切り線 `---` でスライドを分ける

---

# 構文の要点

- **スライド区切り**: 水平線 `---` のたびに新しいスライドになる
- **フロントマター**: 先頭の `---` で囲んだ YAML でテーマなどを指定
- **Markdown**: 見出し・リスト・コードブロックなどがそのまま使える

```markdown
---
marp: true
---
# タイトル
内容
---
# 次のスライド
```
